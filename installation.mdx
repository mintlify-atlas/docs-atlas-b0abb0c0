---
title: "Installation"
description: "Install YT Downloader using the setup installer or Scoop"
---

<Note>
  YT Downloader requires Windows 10 build 17763 (October 2018 Update) or later, on an x64 or ARM64 machine.
</Note>

Choose the installation method that suits you best.

<Tabs>
  <Tab title="Installer">
    <Steps>
      <Step title="Download the installer">
        Go to the [Releases page](https://github.com/TXG0Fk3/YT-Downloader/releases) on GitHub and download the latest `YT-Downloader.Setup.exe`.
      </Step>
      <Step title="Run the setup">
        Double-click `YT-Downloader.Setup.exe` to launch the installer. Follow the on-screen prompts and confirm any permissions requested by Windows.
      </Step>
      <Step title="Launch YT Downloader">
        Once installation completes, YT Downloader appears in your **Start Menu**. Click it to open the app.
      </Step>
    </Steps>

    <Tip>
      The installer bundles FFmpeg, so no additional setup is needed to process and convert downloaded media.
    </Tip>
  </Tab>
  <Tab title="Scoop">
    <Steps>
      <Step title="Install Scoop">
        If you do not already have Scoop, install it by following the instructions at [scoop.sh](https://scoop.sh/). Open **PowerShell** and run:

        ```powershell
        Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
        Invoke-RestMethod -Uri https://get.scoop.sh | Invoke-Expression
        ```
      </Step>
      <Step title="Add the Asterism bucket">
        YT Downloader is distributed through the [Asterism](https://github.com/TXG0Fk3/Asterism) Scoop bucket. Add it by running the following command in **CMD or PowerShell**:

        ```powershell
        scoop bucket add asterism https://github.com/TXG0Fk3/Asterism
        ```
      </Step>
      <Step title="Install YT Downloader">
        Install the app from the Asterism bucket:

        ```powershell
        scoop install asterism/yt-downloader
        ```
      </Step>
      <Step title="Launch YT Downloader">
        After installation, YT Downloader appears in your **Start Menu** under a folder called **Scoop Apps**. You can also run it directly from the terminal:

        ```powershell
        yt-downloader
        ```
      </Step>
    </Steps>
  </Tab>
</Tabs>

## After installation

Once the app is installed, follow the [Quick start](/quickstart) guide to download your first video or audio file.
