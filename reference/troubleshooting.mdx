<AccordionGroup>
  <Accordion title='"FFmpeg not found" error'>
    FFmpeg is not present in the app directory or on the system PATH. The app searches for `ffmpeg.exe` in the YT Downloader installation folder first, then falls back to the `PATH` environment variable. If neither location has it, downloads fail immediately with an error status.

    **Solutions:**

    - **Reinstall using the setup installer.** The installer bundles FFmpeg in the app directory — this is the simplest fix.
    - **Place `ffmpeg.exe` manually in the app folder.** Download FFmpeg from [https://ffmpeg.org/download.html](https://ffmpeg.org/download.html), extract it, and copy `ffmpeg.exe` into the same directory as the YT Downloader executable.
    - **Add FFmpeg to your system PATH.** Extract FFmpeg and add the `bin\` subfolder to your `PATH` environment variable. Open a new terminal and run `ffmpeg -version` to confirm it is detected.

    See the [FFmpeg reference](/reference/ffmpeg) page for detailed setup steps.
  </Accordion>

  <Accordion title="Download fails with an error">
    A download entering the `Error` status can have several causes:

    - The video is **age-restricted**, **private**, or **members-only**. YT Downloader uses the YouTubeExplode library, which can only access publicly available content. Age-restricted and private videos cannot be downloaded.
    - The video is **unavailable in your region** due to geographic restrictions.
    - A **temporary YouTube API issue** or rate limit is interfering with the request.
    - Your **internet connection** was interrupted during the download.

    **Steps to diagnose:**

    1. Try downloading a different, publicly available video to confirm the app is working.
    2. Check your internet connection.
    3. If only specific videos fail, the content is likely restricted and cannot be downloaded.
  </Accordion>

  <Accordion title="Video quality options are limited">
    The quality selector only shows resolutions that YouTube actually provides for the specific video being downloaded. The app queries the stream manifest and filters to MP4-container streams only.

    If a video was uploaded at 720p, higher resolutions such as 1080p or 4K will not appear. Older videos may also lack 60fps variants.

    This is a YouTube limitation — the app cannot offer resolutions that do not exist in the stream manifest.
  </Accordion>

  <Accordion title="The app does not start">
    YT Downloader requires **Windows 10 version 1809 (build 17763) or later**. WinUI 3 is not supported on earlier Windows versions.

    **Steps to resolve:**

    1. Verify your Windows version by pressing `Win + R`, typing `winver`, and pressing Enter.
    2. If your version meets the requirement, try **running as administrator** — right-click the app shortcut and select "Run as administrator".
    3. If the problem persists, **reinstall using the setup installer** to repair any missing runtime dependencies (including the Windows App SDK runtime).
  </Accordion>

  <Accordion title='Download is stuck at "Converting"'>
    The `Converting` status means FFmpeg is actively processing the downloaded streams. This is normal behavior and can take some time for large files or high resolutions.

    - **Wait for it to complete.** Do not close the app while a conversion is in progress. Closing the app will kill the FFmpeg process and leave the output file incomplete. Temporary files are cleaned up automatically after the process ends.
    - If the status does not change after an unusually long time (several minutes for a standard video), cancel the download and try again.
  </Accordion>

  <Accordion title="Settings are reset after restart">
    The app stores its configuration in `%APPDATA%\YT Downloader\config.json`. If this file becomes corrupt — for example after an unexpected shutdown — the app may fail to read it and revert to default settings on each launch.

    **To reset settings manually:**

    1. Close YT Downloader completely.
    2. Open File Explorer and navigate to `%APPDATA%\YT Downloader\` (you can paste this path directly into the address bar).
    3. Delete `config.json`.
    4. Restart YT Downloader. The app will create a new config file with default settings.

    <Warning>
      Deleting `config.json` resets all settings to their defaults, including your default download location and any custom preferences.
    </Warning>
  </Accordion>
</AccordionGroup>
